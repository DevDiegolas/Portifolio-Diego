# Checkpoint: O que gerar hoje?

Bem-vindo ao squad **Portfolio Content Generator**!

Esse squad vai gerar textos profissionais para o portifólio do Diego — bio, descrições de projetos ou textos das seções do site. Tudo baseado no código real do repositório.

## O que você quer gerar hoje?

**Tipo de conteúdo:**

1. **Bio profissional** — quem é o Diego, o que ele entrega, qual o diferencial
2. **Descrição de projeto** — texto técnico com problema, decisão e resultado
3. **Seção do site** — copy para uma seção específica (Arsenal Técnico, Jogos, etc.)
4. **Tudo** — bio + projetos + seções

**Se escolheu Descrição de Projeto:**

- Qual projeto? (pode nomear ou pedir "todos os projetos encontrados")
- Tem algum ângulo específico para destacar? (decisão técnica, game logic, fullstack, etc.)

**Idioma de saída:**

- Português (BR) — padrão
- English — versão EN também
- Ambos — PT-BR e EN

Responda livremente — o squad extrai o contexto da sua resposta e passa para o Pedro Projeto pesquisar no código.
